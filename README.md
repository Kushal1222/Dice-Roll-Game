# 🎲 PyDice - The Dice Roll Game

> Roll the Dice. Claim the Glory!

---

## 📌 About the Project

PyDice is a simple terminal-based Dice Roll Game built in Python.
It was created as part of the Capgemini Python Training Mini Projects.
The game supports two exciting modes — Player vs Player and Player vs Computer.
Each round, players roll a dice and the higher number wins!

---

## 🎮 Game Modes

- **Player vs Player** — Two players take turns rolling the dice
- **Player vs Computer** — Play against PyBot 🤖

---

## ✨ Features

- 🎲 Random dice rolling using Python's `random` module
- 👥 Two game modes — PvP and PvC
- 🔢 Choose number of rounds (1 to 10)
- 🏆 Round by round winner announcement
- 🤝 Tie detection for each round
- 🏅 Final result after all rounds
- 📖 How to Play section in menu
- ❌ Invalid input handling — program never crashes
- 📅 Live date and time shown in menu
- 👋 Clean exit message

## ▶️ How to Run

Open terminal inside the project folder and run:

```bash
cd dice_game
python main.py
```

---

## 📋 How to Play

1. Choose game mode from menu
2. Enter player name(s)
3. Choose number of rounds
4. Press Enter when it's your turn to roll
5. Higher dice number wins the round
6. Player with most round wins is the final winner!
